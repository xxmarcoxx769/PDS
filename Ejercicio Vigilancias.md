#Ejercicio Vigilancias de Examen
##1.Lenguaje ubicuo
| **Término** | **Descripción** |
| :--- | ---: |
| Asignatura | La asignatura para la que se realiza la convocatoria de examen. |
| Profesor Responsable | El profesor que coordina la vigilancia del examen. |
| Profesor Vigilante | El profesor que vigila en el examen |
| Examen | El examen de una asignatura |
| Convocatoria | El periódo de examenes |
| Turno | Mañana o tarde | 
| Solicitud de vigilantes | La solicitud que recibe el secretario de la facultad |
| Titulación | La titulación de la asignatura |
| Fecha de examen | La fecha concreta del examen |
| Secretario de la facultad | El encargado de procesar y validar solicitudes |
| Estado de solicitud | El estado en el que se encuentra la solicitud |
| Aula | El aula donde se realiza el examen |
