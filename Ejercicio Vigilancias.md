# Ejercicio Vigilancias de Examen
## 1.Lenguaje ubicuo
| **Término** | **Descripción** |
| :--- | ---: |
| Asignatura | La asignatura para la que se realiza la convocatoria de examen. |
| Profesor Responsable | El profesor que coordina la vigilancia del examen. |
| Profesor Vigilante | El profesor que vigila en el examen |
| Examen | El examen de una asignatura |
| Convocatoria | El periódo de examenes |
| Turno | Mañana o tarde | 
| Solicitud de vigilantes | La solicitud que recibe el secretario de la facultad |
| Titulación | La titulación de la asignatura |
| Fecha de examen | La fecha concreta del examen |
| Secretario de la facultad | El encargado de procesar y validar solicitudes |
| Estado de solicitud | El estado en el que se encuentra la solicitud |
| Aula | El aula donde se realiza el examen |
## 2.Historias de usuario
Existen tres tipos de usuarios:
* Secretario
* Profesor responsable
* Vigilante
- Como **secretario** quiero asignar un examen a un aula y fecha para planificar su realización.
- Como **profesor responsable** quiero solicitar vigilantes para asegurarme de que el examen de mi asignatura se realiza justamente.
- Como **secretario** quiero revisar y validar solicitudes de vigilancia para asegurar que todos los exámenes tienen los vigilantes que necesitan.
- Como **secretario** quiero lanzar el proceso de asignación automática para que se elijan automáticamente profesores que cubran una vigilancia.
- Como **secretario** quiero validar una asignación automática para asegurarme de que sea correcta.
- Como **profesor vigilante** quiero poder ver la información acerca del examen que tengo que asistir a vigilar y denegar o aceptar la vigilancia.
